02 · Claude Skills

Claude Skills란?

Claude Code에서 / 를 입력하면 나오는 슬래시 명령어입니다. .md 파일 하나로 Claude에게 새로운 능력을 부여합니다.

스킬 목록 확인

Claude Code에서: /    ← 슬래시만 입력하면 목록이 보입니다

스킬 파일 구조

---
name: x402-pay
description: x402 결제를 수행합니다.
---

# 결제 방법

1단계: 유료 엔드포인트에 GET 요청을 보낸다.
2단계: 402 응답을 받는다.
...

Claude는 이 .md 파일을 읽고 거기 적힌 지시대로 코드를 작성해 실행합니다.

스킬은 코드가 아닙니다. Claude에게 보내는 자연어 지시문입니다.

05 · Theory

x402 원리

아이폰 사러 애플스토어 가기

1
아 아이폰 사고 싶다! → 필요한 게 생겼다
2
애플스토어 사이트에 접속 → 어디서 살 수 있는지 확인
3
가까운 오프라인 매장 찾기 → 목록에서 원하는 것 찾기
4
매장에 가서 "아이폰 13 주세요" → 원하는 것을 요청
5
"결제해주세요" → 402 응답: 얼마를, 어디로, 언제까지
6
카드 꽂기 → 서명 생성 + X-PAYMENT 헤더에 담아 재요청
7
카드사 승인 → facilitator가 서명/잔액 검증 + 블록체인 정산
8
아이폰 수령 → 요청한 서비스/퀘스트 응답 수신

x402 대응표

아이폰 사고 싶다
↔
필요한 서비스 발견
애플스토어 접속
↔
GET /llms.txt → /v1/services
"아이폰 13 주세요"
↔
GET /v1/quest/{productId}/{step}
결제 필요 안내
↔
HTTP 402 + accepts[0] (체인/금액/수신자)
카드 꽂기
↔
EIP-3009 서명 → X-PAYMENT 헤더 재요청
카드사 승인
↔
facilitator: 검증 + 온체인 정산
아이폰 수령
↔
퀘스트 문제 응답
핵심: x402는 HTTP 표준 위에서 동작하는 결제 프로토콜입니다. 어떤 언어, 어떤 프레임워크로도 구현할 수 있습니다.
06 · Lab 실습

구멍 채우기

x402-pay/SKILL.md를 에디터로 열고, 아래 힌트를 참고해서 [TODO: ...] 부분을 채워주세요.

#구멍 질문힌트
①몇 번 응답?HTTP 상태 코드 — 이 프로토콜의 이름이기도 함
②응답에 뭐가 있나?아이폰 비유의 "가격표 + 어느 지갑으로"
③to 필드?accepts[0]의 수신자 주소 필드명
④value 필드?accepts[0]의 금액 필드명
⑤validBefore?지금 + maxTimeoutSeconds초
⑥어떻게 인코딩?JSON → 바이너리 → 텍스트로 변환하는 방법
⑦누가 검증?네트워크 정보 테이블에 있음
정확한 답이 아니어도 됩니다. 비슷한 의미면 Claude가 알아서 해석합니다.
